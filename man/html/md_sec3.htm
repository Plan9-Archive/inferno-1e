Security Across the Network

 In conjunction with the use of namespaces in Inferno, the security layer enables applications to access resources on different platforms in a secure manner. An application running on a Unix system, for example, can access resources securely on a system running Windows 95 several hops away. The only requirement is that between the Unix system and the Windows 95 system each connection has been authenticated. Mutually authenticated connections between servers establish trust across the network.
