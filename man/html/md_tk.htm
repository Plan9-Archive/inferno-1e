Limbo/Tk - Graphic Interface Module

The Tk.m module provides primitives for building user interfaces. It is based on Ousterhout's Tcl/Tk. (See Tcl and the Tk toolkit. J. Ousterhout. Addison Wesley, 1994 for a detailed discussion of Tk)

For an introduction to the Limbo implementation of Tk, see Introduction to Tk.
