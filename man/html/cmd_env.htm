Inferno Host O/S Utilities

These environmental utilities are run from the host operating system and not from Inferno. They are as follows:
asm - Dis assembler

emu - Inferno emulator

limbo - Limbo compiler

NOTE:

Multiple options must be entered separately and must be separated by a space. For example:

emu -g600x800 -7 -rusr/release1/inferno
