createsignerkey - command to create signer key on authentication server

lib/createsignerkey name-of-owner [size-in-bits] [file]

Description

 The createsignerkey command creates a private key for the string supplied as the first command argument. The size of the key is given by the second argument (default: 512 bits) and the result is stored in the file named by the third argument (default: /keydb/signerkey).
