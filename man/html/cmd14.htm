limbo - Limbo compiler

limbo [option... ] [file... ]

Description

 The limbo command compiles the named Limbo files into machine-independent object files for the Dis virtual machine. Depending on the options, the compiler may create output files or write information to its standard output.The description of these files produced and the options used in this command are detailed in limbo - Limbo compiler in Chapter 4.
