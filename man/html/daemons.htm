Inferno Daemons

The Inferno daemons are as follows:
cs - connection server daemon

httpd, echo, stats - HTTP server

srv - start server daemons

virgild - connection service for remote clients

NOTE:

The daemons related to security are found in Inferno Security Modules and Utilities in Chapter 9
