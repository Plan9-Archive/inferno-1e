ps - thread status

bind '#p' /prog
ps

Description

 The ps command prints to standard output information about all current threads. The information provided is a formatted version of what is obtained from the status file for each thread.

Caveat

This command requires a populated /prog directory; therefore, there must be a prior binding of the system's #p driver to that directory.

Files

/prog/pid/status
