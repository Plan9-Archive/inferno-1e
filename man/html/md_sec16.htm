manufacture - command to emulate set-top-box-id in ROM

manufacture set-top-box-id

Description

 The manufacture command initializes the file /nvfs/ID with the string set-top-box-id, the command's first argument. This value emulates under directory /nvfs (non-volatile file system) the serial number that the manufacturer of an actual set-top-box would burn into ROM. This value is referenced by the register command.

See Also

register - command to register set-top-box identity with signer

Use of /nvfs/ID to provide an authenticated certificate in /nvfs/default.
